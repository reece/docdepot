The Locus Library is a repository of scientific publications for use by Locus employees.

The Locus Library outside tool is restricted to employees of Locus. Other users will be denied access and are advised to not select this outside tool. The remainder of this document is directed at Locus employees.

The Locus Library is intended to be accessed directly from PubMed using links displayed by NCBI on our behalf. Users may enable the Locus Library service by:

  • Entering PubMed via a URL that specifies the otool parameter:
    http://www.ncbi.nlm.nih.gov/pubmed?otool=locuslibrary
    N.B. This URL will not work correctly until our Outside Tool application with NCBI is approved.
  • Selecting Locus Library as the Outside Tool in their MyNCBI preferences.
