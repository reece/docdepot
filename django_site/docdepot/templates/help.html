The Locus Library is restricted to employees of Locus Development. Other users will be denied access and are advised to not select this outside tool.

The Locus Library

The Locus Library is a repository of scientific publications for use by Locus employees. The Library has two primary interfaces: a shared folder that contains all documents and a simple web service that queries the repository by PubMed id. The web service makes it possible to access the Locus Library directly from NCBI's PubMed web site.

The Shared Folder

All documents are stored in a shared filesystem. These files are completely browsable. Files placed in the incoming folder will be automatically refiled by PubMed id, author, and other metadata. Employees will be given access to the shared folder as needed.

The Web Service

The Locus Library is intended to be accessed directly from PubMed using links displayed by NCBI on our behalf. The library is not browsable via a web interface. Users may enable the Locus Library service by:

N.B. These methods URL will not work correctly until our Outside Tool application with NCBI is approved.
  • Entering PubMed via a URL that specifies the otool parameter:
    http://www.ncbi.nlm.nih.gov/pubmed?otool=locuslibrary
  • Selecting Locus Library as the Outside Tool in their MyNCBI preferences.
